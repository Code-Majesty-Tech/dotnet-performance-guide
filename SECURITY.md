# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this repository, please
report it privately via [GitHub's private vulnerability reporting
feature](https://github.com/Code-Majesty-Tech/dotnet-performance-guide/security/advisories/new).

Please do not open a public issue for security concerns.

We'll acknowledge receipt within 72 hours and keep you informed as we
investigate and address the report.
